# Manifest Lifecycle Diagram Explanation

## Lifecycle Stages

1. **Manifest Creation**
   - Generated during attempted system behavior
2. **Schema Validation**
   - Ensures structural integrity
3. **Semantic Review**
4. **Alignment Review**
5. **Execution Approval/Rejection**
6. **Record in Decision Logs**
7. **Archival / Versioning**

This ensures every action is transparently audited, aligned, and traceable.
